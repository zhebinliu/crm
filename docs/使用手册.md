## 1. 快速开始

### 1.1 登录

访问系统后进入登录页面，填写以下三项信息：

| 字段 | 说明 |
|------|------|
| 工作区 | 您的企业租户标识（如 `demo`） |
| 邮箱 | 您的账号邮箱 |
| 密码 | 登录密码 |

登录成功后自动跳转到**控制台主页**。

### 1.2 界面布局

```
┌──────────┬──────────────────────────────────────┐
│          │  顶部导航（标题 + 操作按钮）            │
│  左侧    ├──────────────────────────────────────┤
│  菜单    │                                      │
│          │  主内容区                             │
│          │                                      │
│──────────│                                      │
│  用户    │                                      │
│  信息    │                                      │
└──────────┴──────────────────────────────────────┘
```

- **左侧菜单**：点击菜单顶部箭头图标可收起/展开侧边栏；收起后悬停图标显示 Tooltip。
- **核心工作区**：控制台主页、销售预测、报表与分析。
- **业务模块**：线索、客户、联系人、商机、报价单、订单、合同、活动等。
- **底部**：当前登录用户信息、进入配置中心（管理员）、退出登录。

## 3. 线索管理

路径：`/leads`

线索（Lead）是尚未确认为客户的潜在销售机会来源。

### 3.1 查看线索列表

- 支持按**状态**（新建 / 跟进中 / 培育中 / 已合格 / 不合格）、**评级**（热 / 温 / 冷）、**行业**、**来源**筛选。
- 支持关键词搜索（姓名、公司、邮箱）。

### 3.2 新建线索

点击页面右上角**「新建线索」**按钮，填写以下字段：

| 字段 | 必填 | 说明 |
|------|------|------|
| 姓 | ✅ | 联系人姓氏 |
| 名 | — | 联系人名字 |
| 公司 | ✅ | 所属公司名称 |
| 职位 | — | 联系人职位 |
| 邮箱 | — | 联系邮箱（状态为"已合格"时必填） |
| 电话 | — | 联系电话 |
| 状态 | — | 默认"新建" |
| 来源 | — | 线索来源渠道 |
| 行业 | — | 所属行业 |
| 线索评分 | — | 0–100 分，辅助优先级判断 |

### 3.3 转化线索

当线索满足资质条件后，点击详情页的**「转化」**按钮，系统将自动：

1. 创建对应的**客户**记录；
2. 创建对应的**联系人**记录；
3. 可选：同步创建关联**商机**；
4. 线索状态标记为"已转化"，不再出现于活跃线索列表。

> ⚠️ 转化操作不可撤销，请确认线索信息完整后再执行。

## 6. 商机管理

路径：`/opportunities`

商机（Opportunity）代表一笔具体的销售机会，是 CRM 的核心对象。

### 6.1 视图切换

商机列表提供两种视图：

| 视图 | 适用场景 |
|------|----------|
| **列表视图** | 快速扫描、搜索、批量操作 |
| **看板视图** | 可视化管理各阶段进展，支持**拖拽**卡片跨阶段 |

### 6.2 销售阶段

系统内置 8 个标准阶段，拖拽或编辑商机时选择：

| 阶段 | 默认胜率 | 预测类别 |
|------|----------|----------|
| 勘探 | 10% | 流水线 |
| 资质评估 | 20% | 流水线 |
| 需求分析 | 30% | 流水线 |
| 价值主张 | 50% | 最佳预期 |
| 提案 | 65% | 最佳预期 |
| 谈判 | 80% | 已承诺 |
| 赢单 | 100% | 已赢单 |
| 丢单 | 0% | 已排除 |

> 修改阶段后，系统自动更新胜率和预测类别。

### 6.3 新建商机

| 字段 | 必填 | 说明 |
|------|------|------|
| 商机名称 | ✅ | |
| 客户 | ✅ | 关联客户 |
| 预计关闭日期 | ✅ | 用于季度预测归属 |
| 阶段 | ✅ | 默认"勘探" |
| 金额 | — | 预估合同金额（元） |
| 负责人 | — | 默认当前用户 |
| 商机类型 | — | 新业务 / 续约 / 现有客户扩展 |

### 6.4 明细行（产品）

在商机详情页可添加产品明细行：

1. 点击**「添加产品」**；
2. 选择产品（从产品目录）；
3. 填写数量、单价、折扣（%）；
4. 保存后自动累加到商机金额。

### 6.5 从商机创建报价单

在商机详情页点击**「创建报价单」**，系统自动将商机明细行、客户信息带入报价单。

## 7. 报价单

路径：`/quotes`

报价单（Quote）是正式向客户提交的价格方案文档。

### 7.1 报价单状态流转

```
草稿 → 审核中 → 已批准 → 已呈递 → 已接受 / 已拒绝
                ↓
             已拒绝（退回草稿）
```

> 金额 ≥ ¥50,000 的报价单需经**销售经理**审批方可生效。

### 7.2 添加产品明细

与商机类似，在报价单详情页可添加/删除明细行，系统自动计算：
- 小计 = 数量 × 单价 × (1 − 折扣%)
- 合计 = 所有有效明细行小计之和

### 7.3 从报价单创建订单

报价单状态为**"已接受"**后，点击**「创建订单」**，系统自动将报价单信息带入新订单。

## 11. 销售预测

路径：`/forecasts`

销售预测模块提供季度维度的配额追踪与团队业绩分析。

### 11.1 视图说明

| 视图 | 适用人员 | 内容 |
|------|----------|------|
| **我的预测** | 所有用户 | 仅展示本人负责的商机与个人配额完成情况 |
| **全团队** | 经理 / 管理员 | 全团队商机汇总 + 团队目标分配表 |

### 11.2 切换季度

点击页面右上角的 `<` / `>` 按钮可切换历史或未来季度，查看不同时间段的预测数据。

### 11.3 预测类别说明

| 类别 | 对应阶段 | 说明 |
|------|----------|------|
| 流水线 | 勘探、资质评估、需求分析 | 所有非丢单商机总金额 |
| 最佳预期 | 价值主张、提案 + 更高阶段 | 乐观情况下的预计收入 |
| 已承诺 | 谈判 + 已赢单 | 本季度基本确定可关闭的金额 |
| 已赢单 | 仅赢单 | 本季已确认的实际收入 |

> 以上四个类别为**累计包含**关系（类似漏斗），"流水线"包含所有非丢单商机，"已承诺"包含谈判和赢单商机。

### 11.4 团队目标分配（经理操作）

1. 切换到**「全团队」**视图；
2. 页面顶部出现**「团队目标分配」**表格；
3. 找到需要设置配额的成员，点击其所在行右侧的**「设置目标」**；
4. 在弹出的对话框中输入该成员的季度配额金额；
5. 点击**「保存目标」**，数据即时写入数据库。

> 只有**管理员**和**销售经理**角色可以为他人设置配额；销售代表只能查看自己的配额（只读）。

### 11.5 商机明细表

预测页下方的商机表格支持按预测类别筛选（全部 / 流水线 / 最佳预期 / 已承诺 / 已赢单），点击商机名称可跳转到商机详情页。

## 13. 管理员功能

> 以下功能仅限**系统管理员**角色使用，入口位于侧边栏底部「⚙️ 配置中心」。

### 13.1 用户管理

路径：`/admin/users`

| 操作 | 说明 |
|------|------|
| 新建用户 | 填写邮箱、显示名称、初始密码，分配角色 |
| 编辑用户 | 修改用户信息或角色 |
| 停用用户 | 不删除记录，仅禁止登录 |

### 13.2 元数据管理

路径：`/admin/metadata`

元数据管理允许在不修改代码的前提下扩展 CRM 对象的字段。

**添加自定义字段步骤：**

1. 在对象列表中选择目标对象（如"商机"）；
2. 点击**「添加字段」**；
3. 填写字段 API 名称（建议以 `__c` 结尾，如 `custom_score__c`）；
4. 选择字段类型：

| 类型 | 说明 |
|------|------|
| 文本（STRING） | 单行文本 |
| 数字（NUMBER） | 整数或小数 |
| 货币（CURRENCY） | 带货币符号的金额 |
| 布尔（BOOLEAN） | 是 / 否 |
| 日期（DATE） | 年月日 |
| 日期时间（DATETIME） | 年月日时分秒 |
| 关联（REFERENCE） | 关联到其他对象的记录 |
| 下拉列表（PICKLIST） | 单选下拉 |

5. 设置是否必填；
6. 保存后字段立即在对应对象的表单中生效。

**页面布局编辑器：**

在对象详情页点击**「编辑布局」**，可拖拽调整字段在详情页的显示顺序与分组。

### 13.3 工作流规则

路径：`/admin/workflow`

工作流规则可在记录满足特定条件时自动执行动作。

**创建规则示例（商机金额 ≥ 10 万自动创建跟进任务）：**

1. 点击**「新建规则」**；
2. 选择对象：`商机`；
3. 触发时机：`记录创建时`；
4. 条件：`金额 ≥ 100000`；
5. 动作：`创建任务`，主题：`大单跟进 - {{record.name}}`，优先级：高；
6. 保存并启用规则。

**可用动作类型：**

| 动作 | 说明 |
|------|------|
| 创建任务 | 自动创建跟进任务 |
| 字段更新 | 自动修改记录字段值 |
| 发送通知 | 向指定用户发送站内通知 |

### 13.4 审批流程

路径：`/admin/approvals/process-config`

**配置报价单审批流程：**

1. 新建审批流程，关联对象选择 `报价单`；
2. 设置进入条件（如：合计金额 ≥ ¥50,000）；
3. 添加审批步骤：
   - 审批人来源：**角色**，选择 `销售经理`；
   - 审批模式：顺序；
   - 拒绝行为：最终拒绝；
4. 设置批准/拒绝后的自动动作（如更新状态字段）；
5. 开启流程。

**处理审批请求：**

路径：`/admin/approvals`

- 查看待我审批的请求列表；
- 点击请求查看详情；
- 点击**「批准」**或**「拒绝」**，可填写审批意见；
- 申请人可在提交后、审批前点击**「撤回」**。

### 13.5 邮件模板

路径：`/admin/email-templates`

创建用于工作流通知的邮件模板，支持变量插值（如 `{{record.name}}`、`{{user.displayName}}`）。

## 14. 角色与权限

系统内置四种角色，权限由高到低：

| 角色 | 代码 | 主要权限 |
|------|------|----------|
| 系统管理员 | `admin` | 所有权限，包含配置中心 |
| 销售经理 | `sales_manager` | 全量读写业务对象，审批权，设置团队配额 |
| 销售代表 | `sales_rep` | 读写自己负责的业务对象 |
| 审批人 | `approver` | 仅审批权限，只读业务数据 |

### 权限说明

- **读（read）**：查看列表和详情；
- **写（write）**：创建和编辑记录；
- **删除（delete）**：软删除记录（数据保留，不展示）；
- **转化（convert）**：将线索转化为客户/联系人/商机；
- **审批（approve）**：处理审批请求。

## 常见问题

**Q：我创建的商机为什么在销售预测里看不到？**

A：销售预测按商机的**预计关闭日期**归属季度。请确认商机的关闭日期落在当前查看的季度范围内（如 Q2 2026 = 4 月 1 日～6 月 30 日）。

**Q：报价单提交审批后无法编辑了？**

A：审批流程启用了"提交后锁定"机制，审批期间记录只读。若需修改，请联系审批人拒绝后重新编辑提交，或联系管理员撤回。

**Q：如何查看某个客户下所有的商机和合同？**

A：进入客户详情页，页面下方的关联区域会列出该客户的所有商机、报价单、订单和合同。

**Q：侧边栏的"业务模块"里没有某个对象？**

A：可能该对象尚未在元数据中创建，或您的角色没有该对象的读权限。请联系管理员确认。

**Q：我的配额是多少，在哪里看？**

A：进入**销售预测**页，切换到「我的预测」视图，页面上方的进度卡片显示当前季度您的配额目标与完成情况。配额由您的直属经理或管理员设置。
